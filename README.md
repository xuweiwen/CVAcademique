# [CVAcademique](https://github.com/xuweiwen/CVAcademique.git)

[![License: LPPL-1.3c](https://img.shields.io/badge/License-LPPL--1.3c-lightgrey.svg)](https://github.com/xuweiwen/CVAcademique/blob/master/LICENSE) [![Latest build of example.pdf](https://img.shields.io/badge/Latest--build-example.pdf-blue.svg?style=flat)](../example-pdf/example.pdf) [![Latest build of example-overlapped.pdf](https://img.shields.io/badge/Latest--build-example--overlapped.pdf-blue.svg?style=flat)](../example-pdf/example-overlapped.pdf)

**CVAcademique** is a clean and customizable LaTeX template designed for creating professional academic CVs. It emphasizes clarity, structured formatting, and ease of use for researchers, students, and professionals who want to present their education, publications, etc., in a polished manner.

## Demo

- [Left-column section-heading version (default)](../example-pdf/example.pdf)
- [Full-width section-heading version](../example-pdf/example-overlapped.pdf)

## Features

- Minimalist and professional layout optimized for academic CVs
- Flexible section-heading layout that adapts to design needs
- Robust publication numbering system
- Automatic date stamping with customizable formats
